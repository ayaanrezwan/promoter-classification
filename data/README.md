# Data

This directory contains the sequence data used for promoter classification. Raw data files are not tracked by git due to size.

## Obtaining the Data

### Positive Set (Promoter Sequences)

**Source**: EPDnew — Eukaryotic Promoter Database  
**URL**: https://epd.expasy.org/epd/EPDnew_select.php  
**Organism**: *H. sapiens*  
**Assembly**: hg38  
**Sequences**: ~16,455 experimentally validated promoters  
**Window**: -249 to +50 relative to TSS (300 bp each)

To download:
1. Go to the EPDnew Select/Download tool (link above)
2. Set Database to "H. sapiens"
3. Check "Select only the most representative promoter for a gene"
4. Click Select
5. On the results page, choose FASTA format, range -249 to +50
6. Save to `data/raw/human_promoters.fasta`

### Negative Set (Non-Promoter Sequences)

Generated synthetically by the pipeline (`src/data_loader.py`). Sequences are GC-content matched to the promoter set to prevent the model from using nucleotide composition as a classification shortcut.

## Directory Structure

```
data/
├── raw/                    # Original FASTA files (download manually)
│   └── human_promoters.fasta
├── processed/              # Generated by the pipeline
│   ├── X.npy              # Feature matrix (n_samples × 64)
│   ├── labels.npy         # Binary labels (1=promoter, 0=non-promoter)
│   └── vocabulary.txt     # K-mer names corresponding to each column
└── README.md               # This file
```

## Processed Data

Running `run_models.py` generates the processed data automatically from the raw FASTA file. If processed data already exists, it is loaded directly to skip re-encoding.
